# Countdown / Hype

Builds anticipation toward kickoff and each tournament stage. Rhythm content
that keeps the feed alive between fixtures.

## Visual
- Big numeral on the grid + a rotating crest wall, or a single hero crest.
- TikTok/Shorts: pixel motion ticking down.

## Caption formula
`[N] days until the only thing that unites all of us.`
or
`[stage] is set. [one-line].`

## Examples
```
30 days until the whole world stops to watch the same thing. 🌐
```
```
the bracket is locked. 32 nations, one trophy, zero suits invited.
```

## Triggers
- countdown milestones: 100, 50, 30, 14, 7, 3, 1 days, kickoff
- group draw day, fixtures release, bracket lock, each round start
