<objective>
Refactor the design upload pipeline to store images in Cloudflare R2 (public URLs) and create print-on-demand products via the Printful API instead of creating Shopify products directly. This is the entire custom code for Phase 3 -- order fulfillment is handled automatically by the Printful Shopify app.

Purpose: Close the loop from generative match design to purchasable print-on-demand merch. The current pipeline stores base64 in D1 and creates Shopify products directly, which Printful cannot sync. The new pipeline goes R2 -> Printful API -> Printful auto-syncs to Shopify -> orders auto-fulfill.

Output: Working design upload endpoint that stores PNGs in R2 and creates Printful sync products. Printful Shopify app configuration (manual) handles the rest.
</objective>

    **POST /upload handler -- complete new flow:**
    1. Parse multipart form data (same fields: image, matchId, title, description)
    2. Validate required fields (image, matchId, title) -- return 400 if missing
    3. Generate `designId` with `crypto.randomUUID()`
    4. Upload image to R2:
       ```typescript
       const arrayBuffer = await image.arrayBuffer();
       const key = `designs/${designId}.png`;
       await c.env.DESIGNS_BUCKET.put(key, arrayBuffer, {
         httpMetadata: { contentType: 'image/png' },
       });
       const imageUrl = `${c.env.R2_PUBLIC_URL}/designs/${designId}.png`;
       ```
    5. Create Printful product (with try/catch, continue on failure):
       ```typescript
       import { createPrintfulProduct } from '../services/printful';

       let printfulProductId = '';
       let syncStatus = 'pending';
       try {
         const result = await createPrintfulProduct(
           c.env.PRINTFUL_API_TOKEN,
           title,
           imageUrl
         );
         printfulProductId = String(result.id);
         syncStatus = 'synced';
       } catch (err) {
         console.error('Printful product creation failed:', err);
         syncStatus = 'failed';
       }
       ```
    6. Ensure designs table exists (same idempotent CREATE TABLE but with new columns including `printful_product_id TEXT` and `printful_sync_status TEXT DEFAULT 'pending'`)
    7. Insert design record with `image_url` (R2 URL, not base64), `printful_product_id`, and `printful_sync_status`
    8. Return JSON response:
       ```typescript
       return c.json({
         success: true,
         designId,
         imageUrl,
         printfulProductId: printfulProductId || null,
         syncStatus,
         note: syncStatus === 'synced'
           ? 'Product created in Printful. It will auto-sync to Shopify.'
           : syncStatus === 'failed'
           ? 'Design saved to R2. Printful product creation failed -- retry or create manually.'
           : 'Design saved. Printful not configured.',
       });
       ```

    **DELETE the `createShopifyProduct` helper function entirely.** Shopify products are now created by Printful's auto-sync, not by our code.

    **Remove the `@shopify/admin-api-client` import** (it was dynamically imported in createShopifyProduct).

    **GET /:matchId handler -- update to include new columns:**
    ```typescript
    'SELECT id, match_id, image_url, printful_product_id, printful_sync_status, created_at FROM designs WHERE match_id = ? ORDER BY created_at DESC'
    ```

    **Add a POST /retry/:designId endpoint** for retrying failed Printful syncs:
    ```typescript
    app.post('/retry/:designId', async (c) => {
      const designId = c.req.param('designId');
      const design = await c.env.DB.prepare(
        'SELECT id, image_url FROM designs WHERE id = ?'
      ).bind(designId).first();

      if (!design) return c.json({ error: 'Design not found' }, 404);
      if (!design.image_url) return c.json({ error: 'No image URL' }, 400);

      try {
        const result = await createPrintfulProduct(
          c.env.PRINTFUL_API_TOKEN,
          `Match Design ${designId.substring(0, 8)}`,
          design.image_url as string
        );
        await c.env.DB.prepare(
          'UPDATE designs SET printful_product_id = ?, printful_sync_status = ? WHERE id = ?'
        ).bind(String(result.id), 'synced', designId).run();
        return c.json({ success: true, printfulProductId: result.id });
      } catch (err) {
        return c.json({ error: 'Printful sync failed', details: String(err) }, 502);
      }
    });
    ```
