    **export.ts**: High-resolution canvas export via WebGLRenderTarget.
    ```typescript
    export async function exportHighRes(
      renderer: THREE.WebGLRenderer,
      scene: THREE.Scene,
      camera: THREE.Camera,
      width: number = 3600,
      height: number = 4800
    ): Promise<Blob>
    ```
    Implementation (from research Pattern 3):
    1. Check `renderer.capabilities.maxTextureSize` -- if less than max(width, height), scale down proportionally and log warning
    2. Create `new THREE.WebGLRenderTarget(width, height, { minFilter: THREE.LinearFilter, magFilter: THREE.LinearFilter, format: THREE.RGBAFormat, type: THREE.UnsignedByteType })`
    3. Save original camera aspect, update camera for export aspect ratio (width/height), call `camera.updateProjectionMatrix()`
    4. `renderer.setRenderTarget(renderTarget)` then `renderer.render(scene, camera)`
    5. `renderer.readRenderTargetPixels(renderTarget, 0, 0, width, height, pixels)` into `new Uint8Array(width * height * 4)`
    6. `renderer.setRenderTarget(null)` -- reset to screen
    7. Restore original camera aspect, call `camera.updateProjectionMatrix()`
    8. Flip Y axis (WebGL bottom-up to canvas top-down): copy rows in reverse
    9. Create `new OffscreenCanvas(width, height)`, get 2d context
    10. Create `ImageData` from flipped Uint8ClampedArray, `putImageData`
    11. Return `canvas.convertToBlob({ type: 'image/png' })`
    12. `renderTarget.dispose()` in finally block

    Export a helper: `export function downloadBlob(blob: Blob, filename: string)` that creates a temporary `<a>` download link.
