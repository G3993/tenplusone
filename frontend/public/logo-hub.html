05Match RulesLAWS / 01–04
Marks are never scaled below 32px in pixel form, never rotated off-axis in 2D application, and never recolored outside the monochrome system (solid = foreground, void = background). The 2px touchline is non-negotiable — it guarantees the mark reads cleanly whether stitched onto fabric, printed at thumbnail scale, or extruded into the live 3D field. Every mark occupies the same 1,024-module pitch, so the full squad shares one optical weight and one structural language.
